# Traefik — Reverse Proxy & TLS Termination

Image: `traefik:v3.7.5`

## Fungsi

Entry point semua HTTP/HTTPS traffic. Auto-discover service via Docker labels.
TLS certificates diperoleh dari step-ca via ACME protocol (auto-renew).

## Port

| Host Port | Fungsi |
|---|---|
| 80 | HTTP — redirect permanen ke 443 |
| 443 | HTTPS — TLS termination |

Dashboard: `https://traefik.dts.system` (basic auth required)

## File Konfigurasi

| File | Status | Keterangan |
|---|---|---|
| `config/traefik.yml.template` | Committed | Template static config |
| `config/traefik.yml` | Gitignored | Hasil render (berisi domain) |
| `config/dynamic/middlewares.yml` | Committed | Middleware & ServersTransport |
| `config/certs/root_ca.crt` | Gitignored | Di-copy dari step-ca |
| `config/auth/htpasswd` | Gitignored | Generated by make init |
| `data/acme.json` | Gitignored | TLS cert storage (chmod 600) |
| `data/access.log` | Gitignored | Access log |

## PENTING

- `acme.json` HARUS chmod 600 atau Traefik akan exit
- `root_ca.crt` HARUS ada sebelum Traefik start (di-copy dari step-ca)
- `traefik.yml` HARUS di-render via `make init` (bukan manual copy)
